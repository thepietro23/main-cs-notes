## ⚠️ IMPORTANT — Format change (Phase 2 Paper 2, IT)
- **2024 cycle:** actual **programming/coding test** on "Parikshak" platform — 3.5 hrs (30 min reading + 3 hrs coding). (see official handout above)
- **2025 cycle:** changed to **MCQ** — "logic-flow completion, debugging, syntax understanding, program dry-run outputs, data analysis", 180 min.
- **Syllabus stayed the same:** Data Structures 40 · Algorithms 30 · OOP 20 · String Manipulation 10.
- → Prepare for **MCQ format** (latest), but the official 2024 handout is still great for question *style* and syllabus.
